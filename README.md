# Audit-of-ChineseHistory-and-the-Universal-Laws-of-Historical-Development
本项目系一项私人研究记录。其立足于中国社会显而易见的客观现象，深入梳理中国漫长历史中被刻意掩盖或尚未被发掘的重要真相，并对中国历史进程中方方面面的表现给出了一致性的合理解释。随后，为验证该解释的普适性，本项目进一步考察了人类古今中外、天下四方的普世历史，并获得了卓越的逻辑拟合。本记录将呈现出迄今为止最具说服力的人类历史发展模型。目前本文稿仅有中文版本，英文翻译将在妥善校对后逐渐放出。(Full text currently in Chinese; English translations will follow periodically after meticulous review.)
